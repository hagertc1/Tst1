# 🛍️ Multi-Tenant Store Management System

نظام متكامل لإدارة مواقع المتاجر الإلكترونية (Landing Pages) يعمل بتقنية **Multi-Tenant SaaS** باستخدام **Firebase**، بحيث يمتلك كل عميل لوحة تحكم مستقلة مع عزل كامل للبيانات.

---

# ✨ المميزات

## 🔐 نظام تسجيل دخول

- تسجيل دخول بالبريد الإلكتروني وكلمة المرور
- Firebase Authentication
- نظام صلاحيات متعدد

---

## 👥 مستويات المستخدمين

### Super Admin

- إنشاء العملاء
- حذف العملاء
- إدارة الاشتراكات
- إنشاء Admin
- تعيين Admin للعملاء
- إدارة جميع المواقع
- إرسال التنبيهات

---

### Admin

- إدارة العملاء المخصصين له
- تفعيل وإيقاف العملاء
- تجديد الاشتراكات
- متابعة الإحصائيات
- إرسال التنبيهات

---

### Client

يمتلك لوحة تحكم خاصة بموقعه فقط ويمكنه إدارة:

- المنتجات
- الأقسام
- العروض
- آراء العملاء
- الأسئلة الشائعة
- المدونة
- SEO
- إعدادات الموقع
- النسخة الاحتياطية
- الاستعادة

# 🔒 حماية البيانات

يعتمد النظام على Multi-Tenant Architecture.

كل عميل يمتلك Tenant خاص به.

ولا يمكن لأي عميل الوصول إلى بيانات عميل آخر.

# 🖼️ رفع الصور

يعتمد النظام على ImgBB API.

لا يحتاج Firebase Storage.

بعد إضافة مفتاح API يمكن رفع:

- المنتجات
- الشعار
- الخلفيات
- آراء العملاء
- صور المقالات
